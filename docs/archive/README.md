# Archived Release Evidence

This directory holds dated evidence documents for past release lines (release
gates, end-to-end testing guides, integration evidence, and validation runs).
They are kept for the historical record and are no longer updated; current
release documentation lives in [`docs/`](../) and current reports in
[`docs/reports/`](../reports/).

- Top-level files: release-gate and v0.5-era evidence documents.
- [`reports/`](./reports/): dated audit and dogfood reports from the v0.4 and
  v0.5 cycles.
