  Gemini lane, not a sandbox/install/verifier infrastructure failure.

The integration runner itself exposed two small harness issues:

- `tests/integration/run.sh` used Bash associative arrays, which break on the
  default macOS Bash 3.2. It now uses a portable `model_for_agent` case helper.
- `tests/integration/check_results.py` counted both failed retry attempts and
  final retry results. It now checks only the latest result per task. The
  regression is covered in `tests/test_integration_check_results.py`.
