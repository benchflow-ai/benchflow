Create a Python script at `/app/reverse.py` that contains a function `reverse_string(s: str) -> str` which reverses the input string.

Requirements:
- The function must accept a single string argument and return the reversed version of that string.
- `reverse_string("hello")` should return `"olleh"`.
- `reverse_string("")` should return `""`.
- The file must be located at `/app/reverse.py`.
